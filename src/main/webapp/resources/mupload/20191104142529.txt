
1. Centos7 ios파일 다운로드 및 부팅 USB만들기

    1) 공식홈페이지 다운로드 https://www.centos.org/download/ DVD ISO 클릭! 다운로드
        위에 있는 4개 사이트가 한국에 있는 사이트 하나만 골라서 다운받으면 됨.
        카이스트 사이트에서 받겠음. http://ftp.kaist.ac.kr/CentOS/7.6.1810/isos/x86_64/CentOS-7-x86_64-DVD-1810.iso

    2) 부팅디스크 만들기 usb준비(4GB)
        * 하나만들어서 돌려가면서 쓰면될것 같음 
        ISO파일 부팅파일로 만들어주는 애플리케이션활용 부팅 usb만듬
        win32diskimager이용하겠음.
        다운로드 사이트 http://sourceforge.net/projects/win32diskimager/
        
        부팅 USB 만드는 과정
        가) 다운로드 후 설치
        나) win32diskimager 실행
        다) iso파일선택, device에서 usb드라이브선택
        라) write클릭
        마) 그럼 알아서 설치가 될것임.

2. 리눅스 설치하기

    1) 부팅모드 설정
        cmos화면에서 부팅모드를 usb로 설정해야지 usb로 부팅이 가능함
        - 부팅모드에서 설정
            컴퓨터 시작화면에서 F2, F8, F11등 cmos에 따라 부팅모드 설정이 다름
            누르면 부팅을 선택을 하는 화면이 나옴 USB를 선택하면 USB로 부팅하게 됨.
        - cmos화면에서 설정
            -- 컴퓨터에서 F8, F11등(컴퓨터마다 다름) 을 누르면 CMOS화면으로 들어갈 수 있음
            -- CMOS로 들어가면 boot메뉴로 진입
            -- 진입하면 아래쪽에 CD, HDD나오는 부분이 나오고 우선순위가 정해져있는데 
            -- usb를 1순위로 설정하면 됨. 화살표키로 이동하여 enter키 누르면 변경이 가능 
            -- 변경후 save & exit나 나가면됨. yes누를것
        -설정하면 재부팅되고 USB로 부팅이 될것임
    
    2) 리눅스 설치화면 GUI방식으로 큰 무리 없이 진행할 수 있음.
        - 언어설정 : 언어는 아래쪽으로 내려가면 한국어 있음 선택
        - 파티션설정(harddisk설정)
            파티션 자동설정과 수정설정으로 나뉨
            자동으로 하면 편하나 여기서는 수정으로 설정하겠음
            리눅스의 폴더별로 용량을 나눌 수 있음. 하지만 우리는 server를 운영하기 위함으로 세분화
            헤서 나누지 않고 기본 /boot, swap, /(root)에 용량을 할당하여 사용하겠음.
        
        설정방법
        가) 설치대상클릭 -> 파티션을 설정합니다 -> 완료
        나) 수동설정면
            - boot설정 : 표준파티션선택 -> + 버튼클릭 -> /boot선택, 용량 500M -> 마운트지점추가 -> 장치유형(표준파티션) 파일시스템(xfs) 확인 -> 완료
            - swap설정 : + 버튼선택 -> swap선택, 용량 16G(램의 두배로 설정) -> 마운트지점추가 -> 장치유형(표준파티션) 파일시스템(swap) 확인 -> 완료
            - /설정(root) : + 버튼선택 -> / 선택, 용량비움(나머지가 다 설정됨) -> 마운트지점추가 -> 장치유형(표준파티션) 파일시스템(xfs) 확인 -> 완료
            - 완료 -> 변경사항적용
        파티션 완료!
             * 파티션요약설명
                /boot : 리눅스부팅할때 필요한 폴더 
                swap : 램의 공간이 부족할때 하드에 있는 공간을 활용하기 위해 확보하는것
                / : 리눅스의 기본폴더로, 윈도우의 c: 드라이브로 생각하면 쉽다.
        다) 네트워크 설정
            네트워크 및 호스트명 클릭 -> 이더넷 켬 선택 -> 완료
            * 이더넷 설정시 enpOs..나오는 명칭이 컴퓨터 네트워크카드명칭임
        라) 설치시작!
            설치시작하면 관리자계정암호를 설정하는 화면이 나옴 클릭
            - 관리자계정 암호설정 암호는 특수문자, 대소문자 포함하여 어렵게 정하는걸 추천함. 
              여기서는 1q2w3e4r%T로 설정하겠음.
               * 공개되면 리눅스의 모든것을 조작할수있기에 주의 -학생들한테 절대로 알려주면안됨!!-
            - 옆에 사용자도 설정할 수 있지만 생략하겠음. 나중에 추가할수있음.

        마) 재부팅 / root 로그인
            - 다설치 후 재부팅하면 부팅선택하는 화면이 나오는데 첫번째 선택.
            - 최소설정으로 설치하였기 때문에 바로 cmd창으로 부팅됨.
            - localhost login : 이 떠있는데 root라고 작성
            - password : 1q2w3e4r%T 작성
              * 오라클 sqlplus와 같이 비빌번호를 입력해도 커서가 움직이지 않으니 참고하길
            - 로그인 성공하면 
              [root@localhost ~]# 이 출력
              
             * 간단설명
               root : 계정명 / @localhost : 컴퓨터명칭 / ~ : 현재폴더 # : 관리자계정
